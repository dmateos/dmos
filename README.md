# mateOS 
Just for learning x86 assembly/cpu stuff.

Only 32 bit for now. 

Inspired by messing around with a little OS on the 6502.

## Project Directories

### 1. `src/`
   - Main kernel files
   - kernel.c contains main entry point

### 2. `src/arch/i686`
   - 32bit x86 CPU specific files
   - 686_init.c contains main x86 entry point
